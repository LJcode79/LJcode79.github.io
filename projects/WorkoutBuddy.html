Title: Workout Buddy

Date: November 21, 2023

Technologies: MongoDB, ExpressJS, ReactJS, NodeJS, HTML, CSS, Railway, Netlify, Postman

IDE: Visual Studio Code

Description: MERN stack application created to keep track of workouts. Uses MongoDB to keep track of workout data, as well as the time when the workout was created. This website utilizes JWT authentication and password decryption for security. The frontend is deployed on Netlify, the backend on Railway.
